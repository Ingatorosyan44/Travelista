# Travelista (static multi-page site)

## Pages

- `index.html`
- `pages/about.html`
- `pages/services.html`
- `pages/gallery.html`

## Run

Open `index.html` in your browser.

If you want a local server (recommended for clean routing and caching), run one of these from the project folder:

- `python3 -m http.server 8080`
- `npx serve`

Then visit `http://localhost:8080`.
